
# Forward Extra 情境计算引擎（AUD/CNH）

## 1. 项目目的

本项目用于构建一个：

> **结构化外汇产品的情境分析计算器（deterministic scenario engine）**

目标是：

- 明确不同市场路径下的**真实现金流变化**
- 理解 Forward Extra 结构在：
  - 正常情况下
  - barrier 触发情况下
  的行为
- 支持交互式输入（React UI）

---

## 2. 本系统“不做什么”（非常重要）

本系统**刻意不包含**：

- ❌ 概率模型（Monte Carlo）
- ❌ 波动率建模
- ❌ 期权定价（Black-Scholes 等）
- ❌ 市场校准

这是：

> ✅ **情境引擎（scenario engine）**

不是：

> ❌ 定价引擎（pricing engine）

---

## 3. 交易结构（核心逻辑）

### 3.1 初始步骤

用户：

- 持有人民币本金：`N`
- 按 spot 汇率 `S0` 换成 AUD



AUD0 = N / S0


---

### 3.2 Forward 合约

- Forward 汇率：`F = 4.70`
- 到期时间：约 2 个月

---

### 3.3 Barrier 条款

- Barrier：`B = 4.59`
- 类型：**路径触发（touch）**

触发条件：


只要观察期内任意时点 spot < B


---

### 3.4 Ratio 条款（已确认，禁止修改）

**核心规则：**


如果 barrier 触发：

总交割 AUD = 2 × AUD0


⚠️ 注意：

- 基准是 **AUD 数量**
- 不是 RMB
- 不能改成其他定义

---

## 4. 情境定义


### 🟢 情境 1：未触发 Barrier

条件：


观察期内未触及 < B


执行：

- 用 AUD0 按 Forward（F）换回 RMB
- 无论到期市场价在哪，结算汇率始终为 F

---

### 🔴 情境 2：触发 Barrier（重点）

条件：


路径中任意时点 < B


执行：


交割 AUD = 2 × AUD0
RMB = 2 × AUD0 × F


---

## 5. 触发后的再配置逻辑（用户策略）

该部分不是产品自带，而是用户策略。

---

### Step 1：原始支付

使用 RMB = N


---

### Step 2：剩余资金


RMB剩余 = 总RMB - N


---

### Step 3：再买回 AUD

按用户设定汇率 `S1`：


AUD2 = RMB剩余 / S1


---

### Step 4：最终结果



净AUD变化 = AUD2 - AUD0


---

## 6. 输入变量

| 变量 | 含义 |
|------|------|
| N | 初始人民币 |
| S0 | 初始 spot |
| F | forward |
| B | barrier |
| S1 | 赎回汇率 |
| r | 年化收益 |
| T | 时间 |

---

## 7. 输出结果

核心输出：

- 初始 AUD
- 触发后总 AUD
- forward 得到的 RMB
- 剩余 RMB
- 再买回 AUD
- 净 AUD 变化

---

## 8. 关键假设（AI 不得破坏）

---

### 8.1 必须是 Deliverable

```

必须是实物交割

```

如果改为 NDF：

> 整个模型失效

---

### 8.2 允许立即反向交易

```

RMB → AUD 可以立即执行

```

否则：

> Step 3 无法成立

---

### 8.3 无交易成本

当前忽略：

- 点差
- 手续费
- 滑点

---

### 8.4 情境是给定的

本系统：

```

不判断是否触发
只计算触发后结果

```

---

## 9. 设计原则

---

### 9.1 现金流优先

模型基于：

```

真实资金流动

```

而不是：

```

抽象收益曲线

```

---

### 9.2 完全确定性

```

输入 → 唯一输出

```

---

### 9.3 无隐含逻辑

所有公式必须：

- 显式
- 可追溯
- 可解释

---

## 10. 常见错误（必须避免）

---

### ❌ 错误 1：把 barrier 当成交价

```

barrier ≠ 可交易价格

```

---

### ❌ 错误 2：混淆 RMB 和 AUD 名义

```

2x 基于 AUD

```

---

### ❌ 错误 3：忽略 S0

初始汇率决定：

- 初始 AUD 数量
- 后续全部规模

---

### ❌ 错误 4：当作套利模型

本质是：

```

路径依赖 + 加仓结构

```

不是：

```

无风险套利

```
